Description

This is an experimental JSON REST API providing METAR and TAF weather data.

Currently provides only raw (undecoded) METARs and TAFs.

See API help for quick intro or Swagger for details.

Examples

Single ICAO location code

/metar/UKLL : get METAR.

/taf/UKLL : get TAF.

/location/UKLL : get location data.

/all/UKLL : get all available data.

Multiple ICAO location codes

/metar?location=UKLL,UKLI,NZSP : get METAR.

/taf?location=UKLL,UKLI,NZSP : get TAF.

/location?location=UKLL,UKLI,NZSP : get location data.

/all?location=UKLL,UKLI,NZSP : get all available data.
